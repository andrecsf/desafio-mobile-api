# 📜 App de Poesia Aleatória

Uma aplicação web simples que exibe poesias aleatórias para incentivar a leitura, reflexão e contato com a literatura.

## ✨ Funcionalidades

- 🎲 Sorteio de poesias aleatórias
- 📖 Exibição organizada do poema (título, autor e conteúdo)
- 🔗 Link para conhecer mais sobre o autor
- 🎨 Interface com foco em tipografia e leitura confortável
- 📱 Layout responsivo (funciona bem em desktop e mobile)

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (EB Garamond)

## 🎯 Objetivo do projeto

O objetivo deste projeto é praticar:

- Manipulação do DOM com JavaScript
- Consumo e exibição de dados dinâmicos
- Estruturação de páginas com HTML semântico
- Estilização com CSS (tipografia, layout e responsividade)

Além disso, o projeto busca proporcionar uma experiência agradável de leitura, simulando o estilo de páginas editoriais e literárias.

## 🎨 Design

- Tipografia serifada para melhor leitura
- Layout centralizado em formato de "card"
- Fundo com imagem temática de elementos literários
- Uso de sombras e espaçamento para criar hierarquia visual
